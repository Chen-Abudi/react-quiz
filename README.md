# ✨ How to Get Started:

## Clone Project:

```bash

git clone https://github.com/Chen-Abudi/react-quiz.git

```

## Enter the project directory:

```bash

cd react-quiz

```

## Install the Dependencies:

```bash

npm install

```

or in short

```bash

npm i

```

## 🚀 Launch:

```bash

npm run dev

```
